## 📖 Short Project Description

**LawGrid** is an AI-powered legal assistant designed to provide intelligent legal reasoning and support using ontology-driven approaches. The system aims to simplify complex legal information by structuring knowledge into meaningful relationships and enabling users to query legal data efficiently.

It leverages artificial intelligence techniques to:

* Interpret legal queries
* Provide relevant legal insights
* Enhance accessibility to legal knowledge
* Support decision-making through structured legal reasoning

This project is developed as part of the partial fulfillment requirements for the degree of  **Bachelor of Engineering in Computer Science and Engineering (Data Science)** .

## ⚙️ How to Run the Project

### 1. Clone the Repository

```bash
git clone <your-repo-link>
cd lawgrid
```

### 2. Install Dependencies

Make sure you have Python installed, then run:

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
python app.py
```

### 4. Access the Project

Open your browser and go to:

```
http://localhost:5000
```
